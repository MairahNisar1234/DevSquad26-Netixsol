# Tip Calculator App 💰

A simple Tip Calculator built using **HTML, CSS, and JavaScript**.
This app helps users calculate the **tip amount per person** and **total amount per person** based on the bill, tip percentage, and number of people.

---

## 🚀 Features

* Enter the **bill amount**
* Select a **tip percentage** (5%, 10%, 15%, 25%, 50%)
* Add a **custom tip percentage**
* Enter **number of people**
* Automatically calculates:

  * Tip amount per person
  * Total amount per person
* **Reset button** to clear all inputs
* Real-time UI updates using JavaScript

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## ⚙️ How It Works

1. The user enters the **bill amount**.
2. The user selects a **tip percentage** or enters a **custom tip**.
3. The user enters the **number of people**.
4. JavaScript calculates:

   * Tip per person
   * Total amount per person
5. The results are updated **dynamically on the screen**.
